# node.js快速编写get/post后端服务接口

今天在做一个需要快速开发完成的Web端的Demo演示程序，需要若干个支撑的后端接口，返回一些简单的演示数据。由于本人在后端方面较为薄弱，并且前端开发较熟练，对JavaScript了解较多，所以计划学习使用node.js构建一个简单的后端，参考了一些博客中的技术分享，终于成功，在此记录一下碰到的一些问题及解决方法。

参考的博客：https://blog.csdn.net/piratest/article/details/54584182 Node.js学习--搭建一个简单的带有get/post接口的服务——海盗roby；

https://www.jb51.net/article/105408.htm 基于node.js依赖express解析post请求四种数据格式——Sherlock_H

对上述各位大牛表示感谢！

首先保证电脑具备node.js环境，新建文件夹，并在文件夹中编写并保存package.json文件，命令行cd至新建的文件夹目录，运行“npm install”命令，系统将自动下载package.json中提到的依赖库，包括express框架，body-parser等。package.json的文件内容如下：

{

    "name": "Express_Server_Demo",

    "version": "1.0.0",

    "description": "demo on express",

    "main": "expressServer.js",

    "scripts": {

        "start": "node expressServer.js",

        "lint": "jshint --exclude node_modules .",

        "test": "echo \"Error: no test specified\" && exit 1"

    },

    "repository": { },

    "author": "djg",

    "license": "ISC",

    "dependencies": {

        "body-parser": "^1.15.2",

        "express": "^4.14.0"

    },

    "engines": {

        "node": "~6.9.3"

    }

}

其中engines根据本人的机器上node环境的版本进行填写。

接下来就是编写实际需要的后端服务代码了，新建一个“expressServer.js”文件，在此文件中编写所需的get、post等接口的具体内容，编写好以后，在程序目录下运行“node .”命令即可启动后端服务程序，可通过postman软件进行接口的访问测试了。

在开发过程中遇到过的问题：最开始参考的代码中，对post接口提交的数据解析部分写法是“var urlencodedParser = bodyParser.urlencoded({ extended: false })”，并将urlencodedParser作为app.post的第二个参数，多次试验均无法获得提交的参数信息，后来查到这种写法是针对提交参数是application/x-www-form-urlencoded格式的，而我常用的提交参数格式是application/json，因此在数据解析部分需写为“app.use(bodyParser.json());”。这样就可以正确解析提交的application/json格式参数了。

另外示例后端服务代码由于与前端程序端口不一致，出现了跨域的问题，我采用了nginx代理的方式统一了前后端的域名和端口，后续再查询是否有更优的解决方法。

expressServer.js示例代码如下：

var express = require('express');//引入express框架，通过此框架快速开发get/post接口

var bodyParser = require('body-parser');//引入body-parser中间件，用于解析接口request中提交的参数

var app = express();

app.get('/get', function(request, response){  //编写get接口

        // 输出 JSON 格式

        data = { 'name':'abc', 'value':'123' };

        console.log(data);

        response.json(data);

    }

);

app.use(bodyParser.json());  //添加bodyParser中间件，通过这一命令可实现对 post中提交的application/json数据格式参数进行解析

app.post('/post',function(request,response){

        var name=request.body.name;

        var data={ "success":true, "data": { "name":"requestName", "value":name } };

        response.json(data);

    }

);

var server = app.listen(8989, function(){

        var host = server.address().address;

        var port = server.address().port;

        console.log("address: %s, port: %d", host, port);

    }

);
